## Quickstart

```shell
npm install -g ultrahope
```

```shell
git add -p
git ultrahope commit
```

Works immediately — no account needed. Free tier includes 5 requests per day; [unlimited plans](https://ultrahope.dev/pricing) are available.

For Jujutsu, run `ultrahope jj setup`, then `jj ultrahope describe`.
